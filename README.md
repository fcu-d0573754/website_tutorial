# Website_Tutorial 



**使用XAMPP建立網頁伺服器，再將程式碼放入指定位置，完成成績分析網站的建立。** 
> *XAMPP是一個把Apache網頁伺服器與PHP、Perl及MariaDB集合在一起的安裝包，允許用戶可以在自己的電腦上輕易的建立網頁伺服器。*
